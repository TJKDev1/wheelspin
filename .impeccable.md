## Design Context

### Users
General-purpose audience — anyone who needs to make a random choice. Friends picking restaurants, teachers selecting students, teams deciding meeting topics. Used in casual, social, and semi-professional contexts.

### Brand Personality
Clean, confident, purposeful. A well-designed tool that feels reliable and intentional, not playful or childish. The interface should evoke calm confidence — users trust the wheel because it looks like it was built with care.

### Aesthetic Direction
Blue-orange brand palette (blue primary, orange accent) with OKLCH color system. Light mode default with dark mode via OS preference. Minimal, functional, no-nonsense. Anti-references: childish game aesthetics, neon/glow effects, cartoonish styling.

### Design Principles
1. **The wheel is the star** — everything else is chrome. Make the wheel visually compelling.
2. **Confidence through restraint** — clean layouts, purposeful color, no decoration for decoration's sake.
3. **Utility first, delight second** — it's a tool that happens to be beautiful, not a beautiful thing that happens to work.
4. **Color with intent** — every color should serve hierarchy, meaning, or brand cohesion.
